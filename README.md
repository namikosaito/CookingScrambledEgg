* Operating System: Linux(Ubuntu) / MacOS Sonoma
* CPU: Intel(R) Core(TM) i9-9900KF CPU @ 3.60GHz / Intel Core i5 @ 2GHz 
* GPU: Nvidia GeForce RTX 2080 SUPER with CUDA 11.8 / Intel Iris Plus Graphics
